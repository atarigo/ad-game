# 英雄救援 (Hero Rescue)

## 路由

`/game/hero-rescue`

## 概念

「99% 的人選錯了！」假廣告經典。畫面顯示英雄面對威脅，玩家從三個道具中選出正確的來拯救英雄。

## 玩法

- 每回合顯示英雄 VS 威脅的場景
- 底部出現 3 個道具選項
- 選對：威脅消滅動畫，進入下一回合
- 選錯：英雄倒下動畫，遊戲結束

## 回合設計

1. VS 火龍 → 劍(✓) / 盾(✗) / 藥水(✗)
2. VS 洪水 → 劍(✗) / 船(✓) / 火把(✗)
3. VS 暴風雪 → 火把(✓) / 冰劍(✗) / 盾(✗)
4. VS 毒蛇 → 解毒劑(✓) / 劍(✗) / 盾(✗)
5. VS 巨石 → 鏟子(✗) / 炸藥(✓) / 繩子(✗)

## 技術

- **PixiJS**: 繪製幾何圖形角色、選項按鈕、動畫效果
- 不需 Matter.js

## 勝負條件

- **勝**: 通過全部 5 回合
- **敗**: 選錯任一回合
