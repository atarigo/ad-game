# 末日生存 (Survival Loop)

## 路由

`/game/survival-loop`

## 概念

寒霜啟示錄風格假廣告經典：控制角色在場景中移動，砍柴打獵 → 搬運物資回收集區 → 收集區產出金幣 → 買便當雇人 → 升級產線。idle/tycoon 生存循環。

## 場景佈局（2D 俯視平面）

地圖為 2D 平面，玩家可往任意方向移動。鏡頭跟隨玩家。

```
     [森林區🌲]          [獵場🦌]
      (左上)              (右上)

  [木材站📦]              [肉品站📦]
    (左中)                (右中)

            [便當店🍱]
             (中下)
```

- **森林區**（左上）：隨機生成樹木
- **獵場**（右上）：隨機出現動物
- **木材站**（左中）：玩家把木頭搬回這裡，有工人時定時產出金幣
- **肉品站**（右中）：玩家把肉搬回這裡，有工人時定時產出金幣
- **便當店**（中下）：用金幣買便當

## 玩家角色

- 點擊/觸碰場景中的任意位置，角色會**走過去**（2D 自由移動）
- 移動速度固定，有簡單的走路動畫（左右搖擺）
- 角色面朝移動方向（左右翻轉）
- **攜帶上限**：同時最多攜帶 5 個物品（木頭/肉/便當共用背包容量，金幣不佔空間）
- 角色頭上顯示目前攜帶的物品圖示與數量

## 玩法

### 第一層：採集原始資源

#### 砍樹

- 角色走到樹旁後自動揮砍，每 0.6 秒砍一下
- 每次扣除樹的 HP（等於斧頭等級×2）
- 樹倒下後掉落 2~3 塊木頭散落在地上
- 角色靠近木頭 → 木頭以**弧線動畫**飛到角色身上，自動拾取（不超過攜帶上限）

#### 打獵

- 角色走到動物旁後自動攻擊，每 0.6 秒打一下
- 每次扣除動物 HP（等於武器等級×2）
- 動物會隨機移動，角色會追蹤目標
- 動物死亡後掉落 2~3 塊肉散落在地上
- 角色靠近肉 → 弧線飛到角色身上拾取

### 第二層：搬運到收集區

- 角色攜帶木頭走到**木材收集區** → 木頭以弧線動畫飛到收集區堆疊
- 角色攜帶肉走到**肉品收集區** → 肉以弧線動畫飛到收集區堆疊
- 收集區顯示目前庫存數量

### 第三層：收集區產出金幣

- **前提**：收集區必須有工人站崗，且有庫存原料
- 木材收集區：每 3 秒消耗 1 木頭，產出 1 金幣
- 肉品收集區：每 3 秒消耗 1 肉，產出 1 金幣
- 產出的金幣堆在收集區旁邊，角色走近自動拾取

### 第四層：買便當、雇人

- 角色攜帶金幣走到**便當店** → 金幣飛出，獲得便當（1 便當 = 3 金幣）
- 角色攜帶便當走到**雇用點** → 消耗 1 便當，雇用 1 名工人

#### 可雇用的工人類型

| 工人       | 雇用點     | 功能                                 |
| ---------- | ---------- | ------------------------------------ |
| 砍柴工     | 森林區     | 自動砍樹，掉落的木頭需玩家或搬運工撿 |
| 獵人       | 獵場       | 自動打獵，掉落的肉需玩家或搬運工撿   |
| 木材站工人 | 木材收集區 | 啟動木材→金幣產線                    |
| 肉品站工人 | 肉品收集區 | 啟動肉→金幣產線                      |

### 第五層：升級

- 角色在對應收集區範圍內時，顯示升級按鈕
- 升級花費金幣，提升產線速度（縮短產出間隔）
- 費用公式：`floor(10 × 1.8^(level-1))`
- 每個收集區 Lv1~10

## 核心循環圖

```
砍樹/打獵 → 撿掉落物 → 搬到收集區 → (需工人) 產出金幣
                                              ↓
                              撿金幣 → 買便當 → 雇工人
                                              ↓
                                    升級收集區（加速產出）
```

## 視覺

- 深綠色背景 + 飄雪粒子（末日感）
- 玩家角色：簡單的小人（頭 + 身體 + 四肢），手持當前工具
- 工人：與玩家類似但顏色不同的小人，站在崗位上有工作動畫
- 掉落物散落在地上有微微彈跳
- 拾取/存放時物品以拋物線弧線飛行
- 收集區有堆疊的物資視覺（木頭堆、肉架等）
- 產出金幣時有閃光特效
- 角色頭上浮動顯示攜帶物品

## 技術

- **PixiJS**: 渲染場景、角色、特效、UI
- 不需 Matter.js

## 勝負條件

- **勝**: 兩個收集區都升到 Lv10
- 無限制時間，自由探索
