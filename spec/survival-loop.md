# 末日生存 (Survival Loop)

## 路由

`/game/survival-loop`

## 概念

寒霜啟示錄風格假廣告經典：砍柴 → 打獵 → 賣資源 → 升級裝備/營地。簡化版生存循環放置遊戲。

## 玩法

### 資源採集
- **砍樹**: 點擊左側的樹木，每次扣除樹的 HP（等於斧頭等級×2），砍倒得木材
- **打獵**: 點擊右側的動物，每次扣除動物 HP（等於武器等級×2），擊殺得肉
- 動物會隨機移動

### 賣出資源
- 賣木材：5+ 木材換金幣（價格隨營地等級提升）
- 賣肉：5+ 肉換金幣（價格隨營地等級提升）

### 升級系統
- **斧頭** Lv1~10：提升砍柴傷害與產量
- **武器** Lv1~10：提升打獵傷害與產量
- **營地** Lv1~10：營地視覺升級 + Lv3 起自動產出資源
- 費用公式：`floor(10 × 1.8^(level-1))`，營地雙倍

### 自動產出
- 營地 Lv3 起，每 2 秒自動產出木材和肉
- 產量 = `(營地等級 - 2) × 2`

## 視覺

- 深綠色背景 + 飄雪粒子（末日感）
- 營地隨等級升級：小屋 → 加營火 → 加旗幟 → 加圍欄
- 砍擊/打擊時有震動效果 + 數字飛出
- 自動產出時營地上方顯示收穫文字

## 技術

- **PixiJS**: 渲染場景、角色、特效、UI
- 不需 Matter.js

## 勝負條件

- **勝**: 三項全部升到 Lv10
- 無限制時間，自由探索
