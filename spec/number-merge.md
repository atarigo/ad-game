# 數字合體 (Number Merge)

## 路由

`/game/number-merge`

## 概念

類 2048 + 物理投擲。從頂部丟下有數字的球，相同數字碰撞合併為加倍數字，目標到達 2048。

## 玩法

- 移動滑鼠/手指選擇投擲位置
- 點擊/觸碰放下球
- 球受重力下落，碰到同數字球合併
- 合併後產生更大的球
- 堆到危險線以上就輸了

## 數字與大小

2 → 4 → 8 → 16 → 32 → 64 → 128 → 256 → 512 → 1024 → 2048

球的半徑隨數字增大，從 18px 到 58px。

## 技術

- **PixiJS**: 渲染球體、數字、邊界
- **Matter.js**: 球的物理（重力、碰撞偵測、摩擦）

## 勝負條件

- **勝**: 合併出 2048
- **敗**: 球堆超過危險線 2 秒以上
