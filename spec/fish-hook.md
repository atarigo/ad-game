# 釣魚大師 (Fish Hook)

## 路由

`/game/fish-hook`

## 概念

假廣告常見的釣魚遊戲。魚鉤從上方垂下，魚在不同深度游動，玩家點擊放下魚鉤，時機對了就能鉤住魚。

## 玩法

- 點擊螢幕放下魚鉤（自動下降）
- 魚鉤碰到魚 → 自動收線 → 得分
- 魚鉤碰到垃圾 → 空手收線
- 魚鉤到底 → 自動收線
- 收線後可再次放鉤
- 60 秒倒數

## 魚種設計

| 種類   | 大小 | 分數 | 深度 | 顏色 |
| ------ | ---- | ---- | ---- | ---- |
| 小魚   | 15px | 1分  | 淺   | 藍   |
| 中魚   | 20px | 2分  | 中淺 | 綠   |
| 大魚   | 25px | 3分  | 中   | 橘   |
| 金魚   | 30px | 5分  | 深   | 金   |
| 稀有魚 | 35px | 10分 | 最深 | 粉紅 |

## 障礙物

- 4 個隨機位置的垃圾（灰色方塊 + X 記號）
- 碰到垃圾強制收線

## 技術

- **PixiJS**: 渲染水面、魚、魚鉤、線
- 不需 Matter.js

## 結束

- 60 秒後顯示釣魚數量與總分
