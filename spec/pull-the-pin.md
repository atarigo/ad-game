# 拔釘子 (Pull the Pin)

## 路由

`/game/pull-the-pin`

## 概念

經典假廣告遊戲。畫面中有水（藍色粒子）被釘子擋住，玩家點擊釘子拔掉它，讓水流到下方的杯子裡。

## 玩法

- 點擊金色釘子將其移除
- 水粒子受重力影響，沿牆壁和通道流下
- 讓足夠的水流入杯子即過關
- 部分關卡有岩漿（紅色粒子），需避免導入杯子

## 技術

- **PixiJS**: 渲染牆壁、釘子、粒子、杯子
- **Matter.js**: 水粒子物理模擬（重力、碰撞、摩擦）

## 關卡

- Level 1: 單一釘子擋住水，正下方是杯子
- Level 2: 左右兩區，一邊水一邊岩漿，各有釘子
- Level 3: 兩層釘子，需依序拔除

## 勝負條件

- **勝**: 杯子內水粒子數量 ≥ 總數的 60%
- **敗**: 所有粒子靜止後，杯子內不夠
