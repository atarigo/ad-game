# Cube War 遊戲術語

## 遊戲概念

- **遊戲 (Game)**: 整個遊戲的總稱，從主選單開始到遊戲結束
- **關卡 (Level)**: 遊戲的整體進程，目前為單一關卡設計
- **回合 (Round)**: 一輪完整的敵人生成和戰鬥循環，包含開始階段、我方階段、敵方階段
- **階段 (Phase)**: 回合內的不同執行階段

## 階段 (Phase)

### 開始階段 (START)
- 關卡開始，敵人隨機出現
- 下方重新隨機出現三個可選方塊

### 我方階段 (PLAYER_TURN)
- 玩家可以拖曳方塊到中間區域放置
- 如果放置達成消除規則，則依序消除對應的行或列上的方塊
- 每塊消除的方塊會攻擊一次正上方區域的敵方方塊
- 如果攻擊完，敵方方塊全滅，這回合結束，進入下一回合
- 如果沒有任何可選方塊可以放置，顯示跳過按鈕進入敵方階段

### 敵方階段 (ENEMY_TURN)
- 每個敵方方塊有冷卻時間，當冷卻結束就會對我方發動攻擊
- 如果我方生命歸零則遊戲結束，回到主畫面
- 每個敵方方塊一次動作後，回到開始階段

## 用詞對照

| 英文 | 中文 | 說明 |
|------|------|------|
| Game | 遊戲 | 整個遊戲流程 |
| Level | 關卡 | 遊戲進程單位（目前為單一關卡） |
| Round | 回合 | 一輪完整的戰鬥循環 |
| Phase | 階段 | 回合內的不同執行階段 |
| Turn | 階段 | 玩家或敵人的行動階段 |
| Stage | 階段 | 遊戲進程的階段（不使用） |

## 代碼中的使用

- `GamePhase`: 遊戲階段枚舉
- `round`: 回合數（數字）
- `phase`: 當前階段（GamePhase 枚舉值）
