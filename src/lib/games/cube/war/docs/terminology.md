# Cube War 遊戲術語

## 遊戲概念

- **遊戲 (Game)**: 整個遊戲的總稱，從主選單開始到遊戲結束
- **關卡 (Level)**: 遊戲的整體進程，每此開始遊戲從1開始，當每回合的敵方全滅則進入下一關卡
- **回合 (Round)**: 一輪完整的敵人生成和戰鬥循環，包含開始階段、我方階段、敵方階段的輪迴
- **階段 (Phase)**: 回合內的不同執行階段

## 場景

1. 主選單場景 - 擁有「開始遊戲」按鈕的場景
2. 關卡場景 - 實際遊玩的場景
3. 結算場景 - 顯示通關層數，擁有「確認」按鈕可以回到主選單場景

## 關卡

1. 主選單場景點擊「開始遊戲」後，進入關卡1。
2. 關卡包含數個回合，直到關卡內的敵方方塊全滅則進入下一關卡。
3. 如果期間我方生命歸零，則遊戲結束，進入結算場景。

## 回合

1. 一個關卡會有數個回合，每個回合都會經過固定階段
2. 階段有：開始階段、我方階段、我方攻擊階段、敵方階段
3. 特殊階段：技能選擇階段（每5關觸發一次）

## 階段

### 開始階段 (START)

此階段，每回合只會進入一次

1. 生成怪物
2. 自動進入我方階段

### 我方階段 (PLAYER_TURN)

1. 產生可選方塊
2. 玩家放置可選方塊
3. 判斷是否可消除（攻擊敵方）
   - 若有消除，進入我方攻擊階段
   - 若無消除，繼續放置方塊
4. 判斷剩餘可選方塊是否可以繼續放置
5. 若剩餘可選方塊都無法放置，顯示按鈕跳出階段，即進入敵方階段
6. 若所有可選方塊都已放置且無攻擊，直接進入敵方階段

### 我方攻擊階段 (PLAYER_ATTACKING)

此為技術實作階段，用於處理子彈飛行和攻擊動畫

1. 子彈從我方區域發射向敵方區域
2. 子彈碰撞到敵人時造成傷害
3. 所有子彈清除完畢後：
   - 若還有可選方塊未放置，回到我方階段
   - 若所有方塊都已放置且敵人全滅，進入關卡完成等待階段
   - 若所有方塊都已放置但還有敵人，進入敵方階段

### 敵方階段 (ENEMY_TURN)

1. 依序走訪每個敵方方塊
2. 減少方塊的行動冷卻時間 -1
3. 若方塊冷卻結束，則上下位移動一下，對我方生命進行攻擊，再重置冷卻時間
4. 所有方塊行動完後：
   - 若敵人全滅，進入關卡完成等待階段
   - 若還有敵人，進入下一回合（回到我方階段）

### 關卡完成等待階段 (LEVEL_COMPLETE_WAIT)

此為技術實作階段，確保所有動畫和子彈都清除完畢

1. 清除所有殘留的子彈和動畫
2. 檢查是否觸發技能選擇（每5關）
   - 若是第5、10、15...關，進入技能選擇階段
   - 否則直接進入下一關卡

### 技能選擇階段 (SKILL_SELECT)

每完成第5、10、15...關時觸發

1. 顯示3個隨機技能供玩家選擇
2. 玩家選擇一個技能
3. 應用技能效果
4. 進入下一關卡
