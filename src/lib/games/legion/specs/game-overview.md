# Legion - 遊戲概述

## 遊戲類型
戰棋回合制策略遊戲

## 核心玩法
玩家指揮一支隊伍在棋盤上與敵人戰鬥，通過策略性地使用技能、道具和移動來擊敗所有敵人。

## 遊戲目標
- 主要目標：消滅所有敵人
- 失敗條件：我方全滅

## 遊戲特色
1. **回合制戰鬥**：明確的階段劃分，玩家和敵人輪流行動
2. **多格子敵人**：敵人可占用 1x1、2x2、2x3 不同大小的格子
3. **戰術深度**：需要考慮站位、攻擊範圍和技能搭配
4. **ECS 架構**：支援複雜的技能、屬性和裝備系統擴展

## 平台
- 手機優先設計
- 固定解析度：360x640
- 支援觸控操作

## 美術風格
- 簡約方塊風格（原型階段）
- 使用不同顏色區分陣營
  - 我方：綠色
  - 敵方：紅色
