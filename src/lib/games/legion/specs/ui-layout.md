# UI 佈局設計

## 螢幕尺寸
- 寬度：360px
- 高度：640px
- 縮放模式：FIT（自動適應螢幕，保持比例）

## 區域劃分

遊戲畫面從上到下分為三個主要區域：

### 1. 敵人區域（上方）
```
位置：頂部
尺寸：6x6 棋盤
格子大小：50x50 px
格子間距：2px
偏移量：Y = 20px
總高度：約 322px
```

**棋盤配置**
- 行數：6
- 列數：6
- 總格數：36
- 格子顏色：深紅色 (#440000)
- 邊框顏色：深灰色 (#333333)

### 2. 我方區域（中間）
```
位置：敵人區域下方 10px
尺寸：3x6 棋盤
格子大小：50x50 px
格子間距：2px
總高度：約 164px
```

**棋盤配置**
- 行數：3
- 列數：6
- 總格數：18
- 格子顏色：深綠色 (#004400)
- 邊框顏色：深灰色 (#333333)

### 3. 操作 UI（底部）
```
位置：螢幕底部
高度：130px
背景顏色：深灰色 (#222222)
內邊距：10px
```

**道具欄（第一行）**
```
格子數量：3
格子大小：50x50 px
格子間距：5px
對齊方式：靠左
顯示文字：「空」
```

**技能欄（第二行）**
```
格子數量：5
格子大小：50x50 px
格子間距：5px
對齊方式：靠左
顯示文字：「無」
行間距：10px（與道具欄的距離）
```

**結束回合按鈕**
```
位置：右側中央
尺寸：100x40 px
顏色：綠色 (#00aa00)
Hover 顏色：亮綠色 (#00ff00)
邊框：2px 綠色
文字：「結束回合」
```

## 棋盤對齊

所有棋盤都採用**水平置中**對齊：
```javascript
const totalWidth = cols * cellSize + (cols - 1) * gap;
const startX = (GAME_WIDTH - totalWidth) / 2;
```

## 顏色規範

```typescript
COLORS = {
  background: '#000000',     // 遊戲背景
  gridLine: 0x333333,        // 格子邊框
  enemyCell: 0x440000,       // 敵人格子
  allyCell: 0x004400,        // 我方格子
  uiBackground: 0x222222,    // UI 背景
  enemy: 0xff0000,           // 敵人單位
  ally: 0x00ff00,            // 我方單位
  itemSlot: 0x444444,        // 道具格
  skillSlot: 0x555555,       // 技能格
  text: '#00ff00'            // 文字顏色
}
```

## 響應式設計

- 遊戲使用固定的邏輯解析度（360x640）
- Phaser Scale.FIT 模式確保：
  - 在小螢幕上自動縮放
  - 在大螢幕上居中顯示
  - 保持長寬比不變形
