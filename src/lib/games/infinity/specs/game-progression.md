# 遊戲進度系統規格

## 地圖等級系統

遊戲共分為五個地圖等級，每級三關，總共 15 關。

### 等級列表

| 等級 | 名稱 | 關卡數 | 關卡編號 | 開放血瓶等級 |
| ---- | ---- | ------ | -------- | ------------ |
| D    | D級地圖 | 3 | 1-3 | D |
| C    | C級地圖 | 3 | 4-6 | D, C |
| B    | B級地圖 | 3 | 7-9 | D, C, B |
| A    | A級地圖 | 3 | 10-12 | D, C, B, A |
| S    | S級地圖 | 3 | 13-15 | D, C, B, A（不開放S級） |

### 關卡結構

```
D級地圖
├── 第1關 → 補給場景 → 第2關 → 補給場景 → 第3關 → 補給場景
C級地圖
├── 第4關 → 補給場景 → 第5關 → 補給場景 → 第6關 → 補給場景
...以此類推
```

## 補給場景（商店系統）

每關結束後進入補給場景，玩家可以購買道具。

### 商品列表

#### 血瓶

| 商品 | 價格 | 開放等級 | 說明 |
| ---- | ---- | -------- | ---- |
| D級血瓶 | 0 | D級以上 | 回復 25% 血量 |
| C級血瓶 | 0 | C級以上 | 回復 50% 血量 |
| B級血瓶 | 0 | B級以上 | 回復 75% 血量 |
| A級血瓶 | 0 | A級以上 | 回復 100% 血量 |

> 注意：S級地圖不開放 S級血瓶，最高只能購買 A級血瓶。

#### 道具欄擴充

| 商品 | 價格 | 限制 | 說明 |
| ---- | ---- | ---- | ---- |
| 道具欄 +1 | 0 | 最多 3 格 | 增加一格道具欄位 |

### 購買規則

1. **血瓶購買**：必須有空的道具欄才能購買
2. **道具欄購買**：最多擴充到 3 格
3. **出售道具**：可以賣掉已購買的道具（價格為 0）

## 場景流程

```
開始遊戲
    ↓
MainScene（戰鬥）
    ↓
勝利 → SupplyScene（補給）→ 下一關
    ↓
失敗 → 遊戲結束
```

## 遊戲狀態管理

### GameState 結構

```typescript
interface GameState {
  // 進度
  currentMapLevel: MapLevel; // D, C, B, A, S
  currentStage: number; // 1-3
  totalStagesCleared: number; // 0-15
  
  // 玩家狀態（跨場景保留）
  playerStats: CharacterStats;
  
  // 貨幣（未來擴充）
  gold: number;
}
```
