# 遊戲畫面配置規格

## 畫面尺寸

| 屬性 | 數值 | 說明 |
| ---- | ---- | ---- |
| 寬度 | 360px | 手機友善尺寸 |
| 高度 | 640px | 手機友善尺寸 |
| 比例 | 9:16 | 直向顯示 |

## 區域劃分

遊戲畫面從上到下分為三個主要區域：

```
┌─────────────────────┐
│                     │
│   敵人區域 (460px)   │
│   5x4 格子系統       │
│                     │
├─────────────────────┤
│                     │
│   玩家區域 (120px)   │
│   玩家角色顯示       │
│                     │
├─────────────────────┤
│                     │
│   UI 區域 (60px)     │
│   回合按鈕、回合數    │
│                     │
└─────────────────────┘
```

### 敵人區域 (Enemy Area)

| 屬性 | 數值 | 說明 |
| ---- | ---- | ---- |
| 高度 | 460px | 自動計算：總高度 - 玩家區域 - UI 區域 |
| 背景色 | #1a4d1a | 深綠色 |
| 格子配置 | 5 欄 x 4 列 | 共 20 個格子 |

#### 敵人格子系統

| 屬性 | 數值 | 說明 |
| ---- | ---- | ---- |
| 格子大小 | 60px x 60px | 正方形 |
| 格子間距 | 12px | 格子之間的間距 |
| 頂部間距 | 40px | 距離畫面頂部的距離 |
| 邊框寬度 | 2px | 格子邊框 |
| 格子顏色 | #143d14 | 深綠色填充 |
| 邊框顏色 | #2d5a2d | 綠色邊框 |

**格子排列：**
- 5 欄（橫向 5 個）
- 4 列（縱向 4 個）
- 總共 20 個格子
- 格子置中對齊

### 玩家區域 (Player Area)

| 屬性 | 數值 | 說明 |
| ---- | ---- | ---- |
| 高度 | 120px | 固定高度 |
| 背景色 | #0d260d | 更深的綠色 |
| 玩家尺寸 | 50px x 50px | 正方形 |
| 玩家顏色 | #00ff00 | 亮綠色 |
| 邊框顏色 | #00aa00 | 深綠色邊框 |
| 底部間距 | 35px | 距離 UI 區域的距離 |

**玩家位置：**
- 水平置中
- 垂直位置：距離 UI 區域上方 35px

### UI 區域 (UI Area)

| 屬性 | 數值 | 說明 |
| ---- | ---- | ---- |
| 高度 | 60px | 固定高度 |
| 背景色 | #0a1a0a | 最深的綠色 |

**UI 元素：**
- 回合按鈕（左下角）
- 回合數顯示（右下角）

## 區域分隔線

| 位置 | 寬度 | 顏色 | 說明 |
| ---- | ---- | ---- | ---- |
| 敵人區域與玩家區域之間 | 2px | #2d5a2d | 綠色分隔線 |
| 玩家區域與 UI 區域之間 | 2px | #2d5a2d | 綠色分隔線 |

## 顏色系統

### 背景顏色

| 區域 | 顏色值 | 說明 |
| ---- | ------ | ---- |
| 敵人區域背景 | #1a4d1a | 深綠色 |
| 玩家區域背景 | #0d260d | 更深的綠色 |
| UI 區域背景 | #0a1a0a | 最深的綠色 |

### 角色顏色

| 角色 | 填充色 | 邊框色 | 說明 |
| ---- | ------ | ------ | ---- |
| 玩家 | #00ff00 | #00aa00 | 亮綠色 |
| 敵人 | #ff0000 | #aa0000 | 紅色 |

### UI 顏色

| 元素 | 顏色值 | 說明 |
| ---- | ------ | ---- |
| 按鈕背景 | #2d5a2d | 綠色 |
| 按鈕懸停 | #3d7a3d | 亮綠色 |
| 按鈕邊框 | #4a8a4a | 更亮的綠色 |
| 文字顏色 | #ffffff | 白色 |

## 視覺層級 (Depth)

| 元素 | Depth | 說明 |
| ---- | ----- | ---- |
| 背景區域 | 0 | 最底層 |
| 敵人格子 | 1 | 格子背景 |
| 角色 | 10 | 玩家和敵人 |
| 血量條 | 5 | 角色上方 |
| 資訊抽屜 | 100 | 最上層 |

## 響應式設計

目前遊戲設計為固定尺寸（360x640），適合：
- 手機直向顯示
- 平板直向顯示（可能會有黑邊）

未來可擴充：
- 自適應不同螢幕尺寸
- 橫向顯示支援
- 桌面版佈局
