## 血量條系統

### 血量條位置

| 屬性 | 數值 | 說明 |
| ---- | ---- | ---- |
| 位置 | 角色頭頂上方 | 距離角色頭部 8px |
| 寬度 | 40px | |
| 高度 | 6px | |
| 邊框寬度 | 1px | |

### 血量條組成

1. **背景條**（灰色）
   - 顏色：`#333333`
   - 邊框：黑色，1px
   - 固定寬度：40px

2. **填充條**（血量顯示）
   - 正常狀態：綠色 `#00ff00`
   - 低血量（≤30%）：紅色 `#ff0000`
   - 寬度：根據血量百分比動態調整
   - 原點：左側中心 `(0, 0.5)`

### 血量條更新

- **即時更新**：每次受到傷害或回復時立即更新
- **動畫**：平滑的寬度變化
- **顏色變化**：血量低於 30% 時變為紅色

### 血量條計算

```typescript
血量百分比 = 當前血量 / 最大血量
填充寬度 = (總寬度 - 2) * 血量百分比
```

## 資訊抽屜系統

### 抽屜外觀

| 屬性 | 數值 | 說明 |
| ---- | ---- | ---- |
| 位置 | 畫面頂部 | 初始在畫面外（上方） |
| 高度 | 200px | |
| 寬度 | 360px | 與遊戲寬度相同 |
| 背景色 | #1a1a2e | 深藍紫色 |
| 邊框色 | #4a4a6a | 藍灰色 |
| 邊框寬度 | 2px | |
| 內邊距 | 12px | |

### 抽屜動畫

| 屬性 | 數值 | 說明 |
| ---- | ---- | ---- |
| 動畫時間 | 200ms | 滑入/滑出時間 |
| 動畫曲線 | Power2 | 緩動函數 |
| 初始位置 | y = -200 | 畫面外（上方） |
| 顯示位置 | y = 0 | 畫面頂部 |

### 抽屜內容

抽屜顯示以下資訊（由上到下）：

1. **角色名稱**（粗體，14px）
2. **血量顯示**（12px）
   - 格式：`HP: {當前}/{最大}`
3. **六屬性**（11px，分兩列）
   - 左列：力量、敏捷、體力
   - 右列：智力、意志、幸運
4. **裝備資訊**（11px，藍色）
   - 武器：`武器: {名稱} (+{攻擊力})`
   - 防具：`防具: {名稱} (+{防禦力})`
5. **道具資訊**（11px，藍色）
   - 格式：`道具: {道具列表} ({數量}/{最大容量})`

### 抽屜互動

**開啟方式：**
- 點擊玩家角色
- 點擊敵人角色

**關閉方式：**
- 點擊抽屜外的任何區域
- 敵人死亡時自動關閉

**互動邏輯：**
- 點擊角色時，阻止事件冒泡到背景
- 使用延遲檢查（10ms）來區分角色點擊和背景點擊

## 角色互動

### 可點擊角色

| 角色 | 可點擊 | 游標樣式 | 功能 |
| ---- | ------ | -------- | ---- |
| 玩家 | ✓ | pointer | 顯示玩家資訊抽屜 |
| 敵人 | ✓ | pointer | 顯示敵人資訊抽屜 |

### 點擊效果

- **視覺回饋**：游標變為手型（pointer）
- **功能**：開啟對應角色的資訊抽屜
- **深度**：角色設置為 `depth: 10`，確保可被點擊

### 點擊衝突處理

使用標記系統防止點擊衝突：
- `clickedOnCharacter`：標記是否點擊了角色
- 延遲 10ms 檢查，讓角色點擊事件先執行
- 如果點擊了角色，不觸發背景點擊事件

## 視覺效果

### 閃爍效果

當角色受到攻擊時，會顯示閃爍效果：

| 角色 | 閃爍顏色 | 持續時間 |
| ---- | -------- | -------- |
| 玩家 | 紅色 (#ff0000) | 100ms |
| 敵人 | 白色 (#ffffff) | 100ms |

**實現方式：**
1. 改變角色填充色為閃爍色
2. 100ms 後恢復原色

### 按鈕懸停效果

| 狀態 | 背景色 | 說明 |
| ---- | ------ | ---- |
| 正常 | #2d5a2d | 綠色 |
| 懸停 | #3d7a3d | 亮綠色 |
| 點擊 | 保持懸停色 | |
