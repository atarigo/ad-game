# 裝備系統規格

## 裝備欄位 (Equipment Slots)

每個角色擁有以下裝備欄位：

| 欄位名稱 | 英文名 | 數量 | 說明 |
| -------- | ------ | ---- | ---- |
| 武器     | Weapon | 1    | 近戰武器，提供攻擊力加成 |
| 身體防具 | Armor  | 1    | 鎧甲，提供防禦力加成 |
| 道具欄位 | Items  | 1-3  | 可攜帶的道具，初始為 1 格 |

## 武器系統 (Weapon System)

### 近戰武器列表

| 武器名稱 | 英文名        | 攻擊力 | 說明           |
| -------- | ------------- | ------ | -------------- |
| 木劍     | Wooden Sword  | +5     | 基礎近戰武器   |
| 鐵劍     | Iron Sword    | +10    | 中級近戰武器   |
| 鋼劍     | Steel Sword   | +15    | 高級近戰武器   |

### 武器效果

- 武器攻擊力直接加算到 `weaponAttack` 屬性
- 影響計算：`近距離攻擊力 = 力量 + weaponAttack`

## 防具系統 (Armor System)

### 鎧甲列表

| 防具名稱 | 英文名      | 防禦力 | 說明           |
| -------- | ----------- | ------ | -------------- |
| 布甲     | Cloth Armor | +1     | 基礎防具       |
| 皮甲     | Leather     | +2     | 中級防具       |
| 鐵甲     | Iron Armor  | +3     | 高級防具       |

### 防具效果

- 防具防禦力直接加算到 `defense` 屬性
- 影響計算：`物理防禦 = 體力 + armorDefense`

## 道具系統 (Item System)

### 道具欄位

- **初始容量**：1 格
- **最大容量**：3 格
- **擴充方式**：待定（未來可透過升級或購買擴充）

### 道具列表

#### 血瓶 (Health Potion)

| 屬性     | 數值 |
| -------- | ---- |
| 名稱     | 血瓶 |
| 英文名   | Health Potion |
| 使用效果 | 回復 50 點血量 |
| 可用次數 | 1 次（使用後消耗） |
| 取得方式 | 角色初始擁有 1 瓶，敵人無 |

### 道具使用規則

1. 道具可在戰鬥中使用
2. 使用後道具會被消耗（從道具欄移除）
3. 血瓶回復量不受血量上限限制（可超過最大血量）
4. 道具欄位有數量限制，需管理空間

## 角色限制

| 角色類型 | 武器 | 防具 | 道具欄位 | 初始道具 |
| -------- | ---- | ---- | -------- | -------- |
| 玩家     | ✓    | ✓    | 1-3 格   | 血瓶 x1  |
| 敵人     | ✓    | ✓    | 無       | 無       |

## 裝備影響計算

### 攻擊力計算

```
近距離攻擊力 = 力量 + 武器攻擊力
```

### 防禦力計算

```
物理防禦 = 體力 + 防具防禦力
```

### 範例

**玩家裝備：**
- 武器：鐵劍 (+10)
- 防具：皮甲 (+2)
- 力量：10
- 體力：10

**計算結果：**
- 近距離攻擊力 = 10 + 10 = **20**
- 物理防禦 = 10 + 2 = **12**
