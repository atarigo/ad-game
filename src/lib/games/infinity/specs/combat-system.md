# 戰鬥系統規格

## 戰鬥流程

戰鬥採用回合制系統，每回合按照固定順序執行。

### 回合執行順序

```
按下「下一回合」按鈕
    ↓
1. 回合數 +1
    ↓
2. 玩家攻擊階段
   ├─ 計算攻擊力
   ├─ 判定暴擊
   ├─ 計算實際傷害
   ├─ 敵人扣血
   └─ 更新敵人血量條
    ↓
3. 檢查敵人存活
   ├─ 若死亡 → 敵人被擊敗，結束回合
   └─ 若存活 → 繼續
    ↓
4. 敵人攻擊階段
   ├─ 計算攻擊力
   ├─ 判定暴擊
   ├─ 計算實際傷害
   ├─ 玩家扣血
   └─ 更新玩家血量條
    ↓
5. 檢查玩家存活
   ├─ 若死亡 → 玩家被擊敗，遊戲結束
   └─ 若存活 → 繼續
    ↓
6. 回合結束回復階段
   ├─ 玩家回復血量和魔力
   ├─ 敵人回復血量和魔力
   └─ 更新雙方血量條
```

## 攻擊階段詳解

### 1. 攻擊力計算

**近距離攻擊力：**
```
攻擊力 = 力量 + 武器攻擊力
```

**範例：**
- 力量：10
- 武器：木劍 (+5)
- 總攻擊力：10 + 5 = **15**

### 2. 暴擊判定

**暴擊機率：**
```
暴擊機率 = 幸運 * 0.1
```

**判定方式：**
- 生成 0-1 之間的隨機數
- 若隨機數 < 暴擊機率，則觸發暴擊

**範例：**
- 幸運：10
- 暴擊機率：10 * 0.1 = **1.0** (100%)
- 幸運：5
- 暴擊機率：5 * 0.1 = **0.5** (50%)

### 3. 暴擊傷害計算

**暴擊傷害倍率：**
```
暴擊傷害倍率 = 1 + (意志 * 0.1)
```

**暴擊後傷害：**
```
暴擊傷害 = 基礎攻擊力 * 暴擊傷害倍率
```

**範例：**
- 基礎攻擊力：15
- 意志：10
- 暴擊傷害倍率：1 + (10 * 0.1) = **2.0**
- 暴擊傷害：15 * 2.0 = **30**

### 4. 實際傷害計算

**傷害公式：**
```
實際傷害 = max(0, 攻擊力 - 防禦力)
```

**防禦力計算：**
```
防禦力 = 體力 + 防具防禦力
```

**範例：**
- 攻擊力：15
- 敵人體力：10
- 敵人防具：布甲 (+1)
- 敵人防禦力：10 + 1 = **11**
- 實際傷害：max(0, 15 - 11) = **4**

### 5. 扣血與更新

1. 扣除目標當前血量
2. 血量不能低於 0
3. 更新目標血量條顯示
4. 顯示閃爍效果（被攻擊方）

## 回復階段詳解

### 每回合回復

**血量回復：**
```
回復量 = 意志
新血量 = min(當前血量 + 回復量, 最大血量)
```

**魔力回復：**
```
回復量 = 意志 * 0.2
新魔力 = min(當前魔力 + 回復量, 最大魔力)
```

**範例：**
- 意志：10
- 血量回復：**10** 點
- 魔力回復：10 * 0.2 = **2** 點

### 回復限制

- 血量回復不能超過最大血量
- 魔力回復不能超過最大魔力
- 回復在回合結束時自動執行

## 戰鬥狀態

### 存活判定

```typescript
isAlive = currentHealth > 0
```

### 戰鬥結束條件

1. **玩家死亡**：`playerStats.currentHealth <= 0`
   - 顯示玩家被擊敗訊息
   - 玩家角色變為灰色
   - 無法繼續回合

2. **敵人死亡**：`enemyStats.currentHealth <= 0`
   - 顯示敵人被擊敗訊息
   - 敵人角色隱藏
   - 敵人血量條隱藏
   - 資訊抽屜自動關閉
   - 無法繼續回合

## 戰鬥數值範例

### 初始狀態

**玩家：**
- 力量：10
- 體力：10
- 意志：10
- 幸運：10
- 武器：木劍 (+5)
- 防具：布甲 (+1)
- 攻擊力：10 + 5 = **15**
- 防禦力：10 + 1 = **11**
- 血量：10 * 10 = **100**
- 暴擊機率：10 * 0.1 = **100%**
- 暴擊傷害：10 * 0.1 = **100%** (2.0 倍)

**敵人：**
- 力量：10
- 體力：10
- 意志：10
- 幸運：10
- 武器：木劍 (+5)
- 防具：布甲 (+1)
- 攻擊力：10 + 5 = **15**
- 防禦力：10 + 1 = **11**
- 血量：10 * 10 = **100**
- 暴擊機率：10 * 0.1 = **100%**
- 暴擊傷害：10 * 0.1 = **100%** (2.0 倍)

### 戰鬥範例

**回合 1：**
1. 玩家攻擊：15 - 11 = **4** 點傷害 → 敵人生存
2. 敵人攻擊：15 - 11 = **4** 點傷害 → 玩家生存
3. 雙方回復：各回復 **10** 點血量

**結果：**
- 玩家血量：100 - 4 + 10 = **106**（超過上限，變為 100）
- 敵人血量：100 - 4 + 10 = **106**（超過上限，變為 100）

**回合 2（假設暴擊）：**
1. 玩家攻擊（暴擊）：15 * 2.0 - 11 = **19** 點傷害
2. 敵人攻擊（暴擊）：15 * 2.0 - 11 = **19** 點傷害
3. 雙方回復：各回復 **10** 點血量

**結果：**
- 玩家血量：100 - 19 + 10 = **91**
- 敵人血量：100 - 19 + 10 = **91**

```

[敵人攻擊] 傷害: 15.0 → 實際: 4.0 | 玩家血量: 96.0/100
[回合結束] 雙方回復
  玩家: HP 100.0/100 | MP 10.0/10
  敵人: HP 100.0/100 | MP 10.0/10
```

## 特殊情況

### 傷害為 0

當攻擊力 ≤ 防禦力時：
- 實際傷害 = 0
- 不會造成任何傷害
- 仍會顯示攻擊動畫和閃爍效果

### 暴擊但傷害為 0

即使觸發暴擊，如果：
```
暴擊傷害 ≤ 防禦力
```
則實際傷害仍為 0。

### 血量溢出

- 回復時血量可以暫時超過上限
- 但會立即被限制為最大血量
- 魔力同理
