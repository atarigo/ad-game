## 🎮 快速開始

### 遊戲流程

1. **初始化**：玩家和敵人建立角色狀態，裝備初始裝備
2. **戰鬥階段**：點擊「下一回合」按鈕進行回合制戰鬥
3. **查看資訊**：點擊角色查看詳細狀態（裝備、屬性、道具）
4. **戰鬥結束**：當一方血量歸零時遊戲結束

### 主要功能

- ✅ 回合制戰鬥系統
- ✅ 角色狀態與屬性計算
